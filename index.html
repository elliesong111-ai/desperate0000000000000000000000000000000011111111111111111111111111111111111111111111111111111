Editopia

脚本驱动的自动剪辑工作流：输入脚本即可生成成片，支持自动配图、BGM、字幕与配音。

16:9 1–5 分钟 多风格模板 本地离线

你可以做什么

  • 脚本一键成片：自动分段、配图、字幕
  • 自动 BGM 混音 + 可选 TTS 配音
  • 多风格输出：节奏快切片 / 解说型 / 教程型 / 混剪型
  • 智能配图：关键词、同义词、分类与图片标签加权

快速开始

python auto-editor/auto_editor.py --script "C:\path\script.txt" --bg-dir "C:\path\images" --bgm "C:\path\bgm.mp3" --output "C:\path\out.mp4"

智能配图（示例）

python auto-editor/auto_editor.py --script "C:\path\script.txt" --bg-dir "C:\path\images" --keyword-dict "C:\path\keywords.json" --category-map "C:\path\categories.json" --image-tags "C:\path\image-tags.json" --output "C:\path\out.mp4"
